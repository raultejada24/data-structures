# Data Structures - C++17

15 data structures implemented in C++17 with templates and manual memory management.

## Structures

- **Lists**: Singly, Doubly, Circular
- **Stacks/Queues**: Stack, Queue, Deque
- **Trees**: BST, AVL, Min/Max Heap, Trie
- **Advanced**: Hash Table, Graph, Disjoint Set, Segment Tree
